# GitHub Publish Notes

This repository has been prepared for GitHub:

- large databases are ignored
- large raw exports are ignored
- a lightweight `results/` bundle is included for review

If GitHub credentials are available on this machine, publish with:

```bash
git remote add origin <your-github-repo-url>
git push -u origin main
```

If the remote repository does not exist yet, create an empty GitHub repository first and then run the commands above.
